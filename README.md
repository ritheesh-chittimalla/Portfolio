# Personal Portfolio Website

This is my **personal portfolio website**, built with **Python Flask**, **HTML**, **CSS**, and **JavaScript**.  
It showcases my journey as an **Artificial Intelligence & Machine Learning student**, my projects, skills, and experience in the field of AI, ML, and software development.  

The portfolio is designed to be **modern, responsive, and interactive** with smooth animations, a clean UI, and easy navigation.  

---

## 🚀 Features

- **Home Section** – A hero banner with animations and an introduction.  
- **About Me Section** – My journey, interests, and motivation as an AIML student.  
- **Projects Section** – Highlighting my work with links, descriptions, and interactive cards.  
- **Skills Section** – Technical skills in AI, ML, Data Science, Python, Flask, and more.  
- **Contact Section** – A working contact form powered by Flask backend.  
  

## 🛠️ Technologies Used

**Backend**  
- Python: Programming language  
- Flask: Lightweight Python web framework  

**Frontend**  
- HTML: Structure of the website  
- CSS: Styling and responsive design  
- JavaScript: Animations and interactivity  

**Styling & Animations**  
- Custom CSS with Flexbox/Grid  
- CSS Animations and Transitions  
- JavaScript for interactive effects (scroll-reveal, modals, theme toggle)
